# Inbox

Use this folder for temporary capture.

Move notes out of this folder after classification:

- source material goes to `wiki/raw/`
- durable concept notes go to `wiki/wiki/`
- decisions go to `wiki/decisions/`
- project status goes to `wiki/projects/`
- meeting notes go to `wiki/meetings/`
- bug and incident records go to `wiki/troubleshooting/`
